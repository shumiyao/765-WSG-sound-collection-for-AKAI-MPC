# 765-WSG-sound-collection-for-AKAI-MPC

AKAI MPCの最新ファームウェア（MPC 3.9以降）で追加された**オシレーター・ウェーブテーブル（Oscillator Wavetable）機能**に対応した波形パックと、すぐに音が鳴らせるデモ用の**キーグループ（Keygroup）データ**の詰め合わせです。

ナムコ**風**WSG（Waveform Sound Generator）などのレトロゲームサウンドを彷彿とさせる、どこか懐かしく味のある音色変化を楽しめるウェーブテーブル＆キーグループを収録しています。

## 概要 (Features)

- **Wavetable Data**: MPC 3.9の新しいWavetableオシレーターに読み込ませて使用できる波形データです。音色が時間やモジュレーションで滑らかに、あるいはステップ状に変化します。
- **Demo Keygroup**: ウェーブテーブルの設定やマッピングがすでに完了しているプリセットデータです。ロードするだけで、すぐにキーボードやパッドでレトロなWSGサウンドを演奏できます。

## 動作環境 (Requirements)

- AKAI MPC Live / Live II / One / One+ / X / X SE / Key 37 / Key 61 など
- **MPC Firmware 3.9 以上**（Wavetable機能が実装されたバージョンが必要です）

## インストール方法 (Installation)

1. このリポジトリからZIPファイルをダウンロード（またはクローン）し、PC上で解凍します。
2. 解凍したフォルダを、MPCで使用している **SDカード** または **USBドライブ / SSD** にコピーします。
   - おすすめの配置場所: 
     - キーグループデータ (`.xpm` / `.xpmp`) やWavetableデータは、ご自身の `Splice` フォルダや `Expansions` 、または任意の `My Samples` などのフォルダへ分かりやすい名前（例: `765-WSG-Collection`）で丸ごと配置してください。
3. SDカード（またはUSBドライブ）をMPC本体に挿入します。

## 使い方 (How to Use)

### 1. デモ用キーグループで今すぐ鳴らす場合
1. MPCの **BROWSER（ブラウザ）** を開きます。
2. インストールしたフォルダ内にある **デモ用のキーグループファイル（`.xty`）** を選択してロードします。
3. トラックのタイプを **KEYGROUP** に設定し、ロードしたプログラム（765-WSG...等）を選択すれば、すぐにパッドや外部鍵盤でレトロサウンドを演奏できます。

### 2. Wavetableオシレーターとして自分で音作りする場合
1. 新規の **KEYGROUP** トラックを作成、または **Drum Program** のプログラムエディットを開きます。
2. MPC 3.9の新しいエディット画面から、オシレータータイプで **Wavetable** を選択します。
3. 本パックに収録されている Wavetable ファイル（`.wav` など）をブラウザからオシレーターに読み込みます。
4. `Wavetable Position` をLFOやエンベロープでモジュレーションすることで、WSG特有の独特な音色変化を作り出すことができます。
